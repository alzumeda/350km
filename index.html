🛣 OpenRouteService API-Key

Für Routen-Isochrone wird ein kostenloser API-Key benötigt.

Registrieren auf openrouteservice.org → Dashboard → Token kopieren.

Standard-Plan: 500 Req/Tag · 20 Req/Min · max 120km Distanz
Mehr benötigt? Collaborative Plan (NGO/Uni/Behörden, kostenlos) oder eigener ORS-Server

⚡ OpenChargeMap API-Key

Für genauere Ladesäulendaten (Echtzeit-Status, verif. Leistung) wird ein kostenloser API-Key benötigt.

Registrieren auf openchargemap.org → API Key anfordern (kostenlos, kein Limit).

Ohne Key: Fallback auf OpenStreetMap-Daten.

⚡ OpenChargeMap API-Key

Für präzisere Ladesäulendaten mit Echtzeit-Status wird ein kostenloser API-Key benötigt.

Registrieren auf openchargemap.org → API Key kopieren.
